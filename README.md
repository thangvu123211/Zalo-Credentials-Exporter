# Zalo Credentials Exporter

Chrome Extension dùng để export credentials từ Zalo Web.  
Extension **chạy hoàn toàn local**, không gửi dữ liệu ra ngoài.

> ⚠️ Không liên kết, không đại diện cho Zalo.  
> Dự án phục vụ mục đích học tập & sử dụng cá nhân.

---

## ✨ Tính năng

- Export cookies / credentials từ Zalo Web
- Lưu file về máy người dùng
- Không có backend
- Không theo dõi, không thu thập dữ liệu
